# Schrankensimulator (Gate Simulator)

A small local web app that simulates rail crossing gate closures based on nearby public transit departures (RMV) and OpenStreetMap data.

Drag the 3D marker to a crossing (“gate”), connect, and see upcoming closure windows.

Timing of closure intervalls 

## Requirements
- Node.js 18+ recommended
- An RMV API key (you must use your own key)
